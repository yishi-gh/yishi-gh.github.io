人工智能基础概念

人工智能是研究如何让机器表现出类似人类智能行为的学科，核心能力包括感知、学习、推理、规划、决策、自然语言理解和问题求解。

  • 人工智能(Artificial Intelligence)：最大范围，目标是让机器具备智能。

K-近邻分类 KNN

概念

KNN 是有监督算法，可以用于分类也可以用于回归。其核心思想是：如果一个样本在特征空间中的 K 个最相似（使用特征空间内的距离度量）的样本中的大多数都属于某一个类别，则该样本也属于这个类别。

当 KNN 被用于分类时，又可分为投票法和距离加权法。
- 投票法：最近的 K 个邻居中哪个分类邻居最多，新样本就属于哪类。
- 加权法：给每个邻居赋予一个权重（通常与距离成反比），累加权重确定分类。

类似地，当 KNN 被用于回归问题时，也可以对 K 个邻居的特征向量求平均或加权。

下面主要介绍 KNN 用于分类时的投票法。

算法步骤

  1. 计算测试点到所有训练点的距离。
  2. 距离从小到大排序。
  3. 取前 K 个邻居。
  4. 多数投票得到类别。

关键

KNN 的关键主要在于以下三点：

  1. K的选择：K代表将K个近邻纳入考虑，影响模型泛化能力，需要交叉验证确定，
  2. 距离度量的选择：受数据物理意义影响，常用的有欧氏距离/曼哈顿距离等，
  3. 特征的构造：需要对数据进行标准化或归一化以统一影响。

此外，过多的特征输入 KNN 时，会导致距离集中现象，使得距离的表征能力下降，需要构造特征函数降维。

K-均值聚类 KMeans

概念

KMeans 是无监督聚类算法，不依赖于数据自带标签，而是自发地把样本分成 K 个簇，使样本到所属簇中心的距离平方和最小。

算法步骤

  1. 初始化 K 个中心。
  2. 分配：每个样本归到最近中心。
  3. 更新：重新计算每个簇样本均值，作为新的中心。
  4. 重复 2、3，直到中心不变或达到最大迭代次数。

缺点

  • 需要预设 K，需要反复实验，
  • 对初始中心设置敏感，从而陷入局部最优解，
  • 对异常离群值敏感，可能导致中心被拉的很远，
  • 对簇大小、密度差异敏感，预设簇大小、密度相近。

需要注意的是，K-means 和 KNN 不同，前者 K 代表 K 类，后者 K 代表 K 个近邻。

朴素贝叶斯分类

概念

朴素贝叶斯（Naive Bayes）是一种基于贝叶斯定理与特征条件独立假设的经典有监督分类算法，核心思想是利用先验概率计算后验概率。其中朴素指的是假设特征相互独立，这是一个非常强的假设。虽然现实中往往不成立，但可以简化计算。

其公式表示：在观测到一组特征 后，计算它属于类别 的后验概率。以判断垃圾邮件为例，计算时，使用先验概率 （历史垃圾邮件的比例）乘以似然概率 （如果是垃圾邮件，这些词同时出现的概率）来调整方向，再除以特征的总概率 进行标准化，从而得到后验概率 （在含有这些关键词的条件下，该邮件是垃圾邮件的概率）。

全连接神经网络 DNN

拟合的概念

拟合是实现分类的手段。如果数据可分，就存在一个边界能够把数据分成两类，拟合则是尽量去描述这个边界，最小化决策边界和真实边界的偏差。如果数据在当前特征空间是线性不可分的，通常有两条路：
- 映射到高维特征空间，转化为线性可分：如 SVM
- 扭曲、拉伸、投影为另一个低维嵌入空间（流形）：如 DNN

基本结构

一个标准的 DNN 通常由三部分组成，层与层之间采用全连接的方式，即相邻两层之间每个神经元都相连。

  • 输入层（Input Layer）：负责接收原始数据。数据的每个特征对应一个输入节点。
  • 隐藏层（Hidden Layer）：隐藏层的每一层都在对数据进行特征提取和抽象——前几层可能只能识别简单的“线条”，中间几层能识别“形状”，越往后越能识别复杂的“概念”（如眼睛、人脸）。
  • 输出层（Output Layer）：产生最终的预测结果。节点数由类别数决定。

对于单个神经元，其作用是对于输入的信号先线性变换，再非线性激活。每个神经元内部都有一个线性函数和一个激活函数。

加权求和的权重 代表这个信号的重要程度，偏置 调整触发难易度。这一步的结果会被送入一个非线性的激活函数：

激活函数的作用是为网络注入非线性的特征表达能力。如果没有激活函数，无论多少层线性网络叠加，最终也还是一个线性网络。相较于传统机器学习，非线性网络为 DNN 提供了强大的特征表征能力，使现代机器学习摆脱了特征工程，从而使端到端学习变成了可能。

  • Sigmoid：。将输出映射到 区间，非常适合作为二分类的输出层；但在深层网络中极易引发严重的梯度消失，且计算量较大。

卷积神经网络 CNN

核心思想

DNN 如果用于图像处理，需要将二维像素展开为一维向量，这不仅完全破坏了像素的空间关系，也导致了参数爆炸。CNN 采用以下三大机制提取图像的空间特征：

  • 局部连接：卷积核只看局部区域。
  • 权值共享：同一卷积核在整张图上滑动。
  • 池化降采样：降低尺寸，增强鲁棒性。

从构成上看，分为：

  • 卷积层：负责提取特征。同一个卷积核在图片上移动时，内部权重不变，从而避免了参数爆炸，
  • 激活层：使用激活函数，获得非线性拟合能力，
  • 池化层：保留主要特征的同时，缩小特征图的尺寸，实现降维与特征压缩。

卷积

卷积的本质是一个滑动窗口在二维矩阵上的加权求和。从离散的角度出发（如 CNN），就是对图像对应的数组上一个小的窗口进行加权求和；而从连续的角度看（如信号与系统），就是对线性时不变系统进行滑动加权积分。

  1. 覆盖：将卷积核覆盖在输入矩阵的某一个局部区域上。
  2. 点乘：将卷积核上的每个数字，与输入矩阵对应位置的像素值相乘。
  3. 求和：把这些乘积全部相加，再加上一个偏置，得到的值就是输出矩阵中对应位置的唯一结果。

扫描完成后，窗口向右移动一个步长 S，重复上述计算，直到整张图都被扫描完。

池化

池化也称下采样，同样使用一个滑动窗口在特征图上移动，但是只保留部分特征，比如：
- 最大池化：窗口内取最大值，保留最强烈的特征。
- 平均池化：窗口内取平均值，保留平均特征。

池化的关键作用是：
- 对特征降维：如采用 2×2，步长为 2 的窗口，最终得到的特征图长和宽都是原来的1/2，大大降低计算量。
- 带来平移不变性：保留强烈特征，同时增强对图像细微变化的鲁棒性。
- 扩大感受野：层数越深，一个像素映射的原始窗口越大，从而能捕捉到更深的语义信息。

P <-> Q P 当且仅当 Q
∀x 对所有 x
∃x 存在 x

优先级：¬ > ∧ > ∨ > -> > <->。

谓词逻辑

谓词用于表达对象属性或对象关系。

例：

  • STUDENT(Wang)：Wang 是学生。
  • TEACHER(father(Wang))：Wang 的父亲是教师。
  • “所有教师都有学生”：∀x(TEACHER(x) -> ∃y STUDENT_OF(y,x))。

产生式系统

规则形式：

产生式系统组成：

  • 规则库。
  • 综合数据库/工作记忆。
  • 推理机/控制系统。

产生式系统使用规则库存储规则，数据库存储事实，推理机匹配规则。

推理方式：

  • 正向推理：由事实推出结论，数据驱动。
  • 逆向推理：由目标反查事实，目标驱动。
  • 混合推理：两者结合。

框架表示

框架用于表示结构化知识。类似 CPP 中类内定义的概念。

框架推理常通过继承、匹配、填槽实现。

推理与可信度 CF

推理可以分为：

  • 演绎推理：从一般规则推出特殊结论。
  • 归纳推理：从个别事实总结一般规律。
  • 确定性推理：真假明确。
  • 不确定性推理：结论带概率、可信度或模糊性。

合一与归结

  • 合一：找变量替换，使两个谓词表达式一致。
  • 置换：变量到项的映射，如 {x/a}。
  • 归结：用互补文字消去证明结论。

归结反演：要证明 G，把 ¬G 加入知识库；若推出空子句，则 G 成立。

搜索算法与遗传算法

状态空间

状态空间法用“状态”和“操作”表示问题。常写作：

  • S：状态集合。
  • F：操作集合。
  • S0：初始状态。
  • G：目标状态集合。

从而可以表示：

  • 解：从初始状态到目标状态的操作序列。
  • 最优解：代价最小的路径。

Open 表和 Closed 表

  • Open 表（待探索）：已生成但未扩展节点。
  • Closed 表（历史足迹）：已经扩展节点。

盲目搜索

盲目搜索可以分为 广度优先BFS 和 深度优先DFS。

  • BFS：按层扩展结点，保证最优解，但空间复杂度极高。
  • DFS：沿一条路走到叶子节点再回头，不保证最优解，对于环路和无限深可能陷入死锁。

启发式搜索

估价函数：

- g(n)：从初始节点到 n 的实际代价。这部分是后验的，已经发生的，确定的。
- h(n)：从 n 到目标节点的估计代价。

从而我们有：

  • A 算法：每次扩展 Open 表中 f(n) 最小的节点。
  • A* 算法：若 h(n) 不高估真实代价，即 h(n)<=h*(n)，则可保证最优。

二者的关键区别在于：A 算法不保证解的最优性，只保证可行。对 过高的估计可能会导致对该路径的过分悲观。而 A* 算法保证对每条路径都保持乐观，永远低估其代价，从而保证不放过任何一个可能。同时，搜索越向后进行，实际代价累计越高，当实际代价已经超过估计代价时，A* 就会对这条路径剪枝，从而避免退化为完全的遍历。

遗传算法

遗传算法是模拟自然进化的概率搜索算法。

概念 含义
种群 多个候选解集合
个体 一个候选解
染色体 个体的编码串
基因 染色体的基本单位
适应度函数 衡量解好坏
选择 好个体更可能保留
交叉 两个个体交换部分基因
变异 随机改变基因

基本流程：编码候选解 -> 初始化种群 -> 计算适应度 -> 选择 -> 交叉 -> 变异 -> 重复直到终止。
