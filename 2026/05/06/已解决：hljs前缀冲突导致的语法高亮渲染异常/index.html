问题

代码块的复制、折叠、行号等都正常，但语法高亮异常，只显示主题色方框。

配置

  • Hexo 8
  • hexo-theme-keep 4.2.5
  • hexo-renderer-pandoc 0.5.0
  • highlight.js 11.11.1

解决

修改 highlight.hljs 为 false

分析

highlight.hljs: true 会导致 Hexo 输出 hljs-* 类名。但这不符合 Keep 主题期望接受的样式映射，最终导致高亮渲染异常。

此外，如果没有开启 highlight.auto_detect ，请注意手动填写的语言种类是否能被识别。如 MySql 要写成 Sql ，否则也可能识别异常。
